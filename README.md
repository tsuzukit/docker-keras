# Commands

## Build

```
$ sh script/build/all.sh
```

## Start containers

```
$ sh script/dev/start.sh
```

## Finish containers

```
$ sh script/dev/stop.sh
```

## Enter inside of container

```
$ sh script/dev/enter.sh
```
